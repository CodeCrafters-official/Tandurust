# Tandurust

An AI-powered triage and patient monitoring platform designed to improve healthcare accessibility in rural areas.

---

## Overview

Tandurust is a healthcare-focused application that leverages Artificial Intelligence to assist in patient prioritization, monitoring, and accessibility. The system aims to bridge the gap in healthcare services, especially in rural and underserved regions.

---

## Features

- AI-based triage for patient prioritization  
- Medicine stock tracking system  
- IVR and voice support for accessibility  
- Multilingual support for diverse users  
- Government scheme suggestions for patients  

---

## Tech Stack

### Frontend
- Flutter
- Dart

### Backend
- Python (Flask)

### Database
- NeonDB / MySQL

---

## How It Works

1. Patients input symptoms or interact via voice/IVR  
2. AI model analyzes inputs and prioritizes cases  
3. System tracks medicine availability  
4. Provides relevant healthcare suggestions and alerts
